Top 10 Flowers Website

A Journey in Typography and Layout Grids.

Project Overview

  • Role: UX/UI Designer
  • Team: Independent Project
  • Timeline: 4 weeks
  • Tools: Figma, Adobe Illustrator

  • As part of my typography class (IDX104), I created a responsive website design to showcase skills in typography and layout. The project criteria stated that to be successful, I would need to use 10  unique typefaces, interpret a good balance of layouts and grid system. I created The website "Floral Favourites", displaying unique layouts and style for each flower

  • The Challenge

    Problem Statement: Creating a floral website that uses 10 unique typefaces requires balancing aesthetics with usability through cohesive typography, effective grids, and responsive layouts.  This project addresses these issues by developing a visually consistent website with intuitive navigation and custom UI icons, optimized for desktop and mobile.

  • Outcome

    A responsive website showcasing 10 of my favourite  flowers, designed to offer an engaging and informative user experience across desktop and mobile devices. The site features unique typography for each flower, a cohesive grid-based layout, custom UI icons and maintaining visual consistency with the floral theme.  This project demonstrates skills in layout design, typography, responsive web design, and icon creation, while balancing creativity with usability within a limited timeframe.

  • Research & Inspiration

    To begin, I researched floral websites, looking  at color, typography, and layout trends. I was particularly drawn to the use of grid systems to structure content, realizing breaking the grid creates visual emphasis. This later  informed my mood board, which featured soft pastel colors, elegant fonts, and high-quality imagery – all essential for capturing the personality of each flower.

  • Takeaways:

  • I had to find fonts that show elegance and timelessness.
  • Pastel colors show softness and calmness.
  • High-quality imagery is essential for showcasing the beauty of flowers.

  • Design Process

    Content Development

    I selected ten flowers based on their visual characteristics and symbolism. For each flower, I wrote and had Ai generated text that gave descriptive placeholder text highlighting its unique features while adhering to proper grammar and sentence structure as per the brief.

  • Wire-framing & Layout Grids

    Using low-fidelity wireframes, I planned the layout for both desktop (1000px wide) and mobile (375px wide). A 6-column grid system for desktop and a 5 column-grid system for mobile which created alignment across all pages while maintaining flexibility for breaking the grid layout.

  • Low fidelity 

  • Mid fidelity

  • Typography

    Part of the criteria was to assign unique fonts to reflect the a shared body font across all pages.

  • Icon Creation

    I designed eight custom UI icons using Adobe Illustrator and figma. Later picking five I felt suited and followed the same visual language as the website—rounded edges, clean lines.

  • Challenge Creating icons that were simple enough to be recognizable at small sizes while still maintaining a cohesive visual language that fit the floral theme of the website

  • Overall Challenges

    • Balancing unique typography for each flower with overall consistency was challenging but resolved by using a shared body font across all pages.
    • icons that were both visually appealing and functional across different screen sizes and contexts within the website.
    • ensure appropriate type size and corresponding leading is used and typographical hierarchy is used effectively to guide the reader through content.

  • Final Design

    The final Top 10 Flowers Website provides an elegant user experience that highlights each flower's beauty while keeping a seamless design across devices.

  • Key Features:

    • Custom UI Icons: Rounded-edge icons enhance navigation while aligning with the floral theme.
    • Unique Typography: Font pairings reflect each flower's personality while maintaining readability.
    • Visual Hierarchy: Effective use of headers ensures users can easily navigate content.
    • Responsive Layouts: Images and text adapt seamlessly to different screen sizes.
    • Consistent Design Language: aligned visuals create a cohesive experience.

  • Reflection

    This project allowed me to learn and test my skills in layout design, typography systems, responsive web development, and icon creation. The final product met all assignment criteria while demonstrating my ability to balance creativity with functionality.

    Results

    • Successfully designed 10 visually distinct yet cohesive pages of mobile and desktop.
    • Created 5 custom UI icons.
    • Learned how to create a responsive website for both desktop and mobile.

    What I Would Do Differently:

    • Explore subtle animations or hover effects to enhance interactivity.
    • Experiment with additional icon styles to further elevate navigation.
